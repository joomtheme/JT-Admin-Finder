# JT Admin Finder Demo

This page shows the main administrator screens and usage flow of **JT Admin Finder** for Joomla Administrator.

JT Admin Finder adds a keyboard-driven command palette to Joomla Administrator. It can be opened with **Ctrl + K** on Windows/Linux or **Cmd + K** on macOS. It also supports Joomla’s native keyboard shortcut flow with **J then K**.

---

## 1. Administrator Component Dashboard

The JT Admin Finder administrator component provides a simple dashboard with usage information, plugin settings access, permissions guidance, and project support links.

![JT Admin Finder administrator component dashboard](https://raw.githubusercontent.com/joomtheme/JT-Admin-Finder/main/docs/demo/images/jt-admin-finder-component.png)

---

## 2. Command Palette Search

Press **Ctrl + K** on Windows/Linux or **Cmd + K** on macOS to open the command palette.

Start typing to search administrator menu links and common Joomla backend destinations. Use the keyboard to navigate results and press **Enter** to open the selected item.

![JT Admin Finder command palette search modal](https://raw.githubusercontent.com/joomtheme/JT-Admin-Finder/main/docs/demo/images/jt-admin-finder-modal-search.png)

---

## 3. Joomla Dashboard Shortcut Hint

When enabled, JT Admin Finder displays a dashboard shortcut hint in Joomla Administrator.

The shortcut hint helps administrators discover the command palette quickly from the Joomla dashboard.

![JT Admin Finder shortcut hint on Joomla dashboard](https://raw.githubusercontent.com/joomtheme/JT-Admin-Finder/main/docs/demo/images/jt-admin-finder-shortcuts.png)

---

## 4. System Plugin Settings

The system plugin controls the command palette behavior inside Joomla Administrator.

Available settings include enabling or disabling the command palette, setting the maximum number of search results, and choosing whether Joomla core administrator links should be included.

![JT Admin Finder system plugin settings](https://raw.githubusercontent.com/joomtheme/JT-Admin-Finder/main/docs/demo/images/jt-admin-finder-system-plugin.png)

---

## 5. Dashboard Hint Setting

The dashboard hint can be enabled or disabled from the plugin settings.

This allows site administrators to decide whether the shortcut reminder should be shown on the Joomla dashboard.

![JT Admin Finder dashboard hint plugin setting](https://raw.githubusercontent.com/joomtheme/JT-Admin-Finder/main/docs/demo/images/jt-admin-finder-system-plugin-dashboard-hint.png)

---

## Keyboard Shortcuts

| Shortcut     | Platform / Context                     |
| ------------ | -------------------------------------- |
| `Ctrl + K`   | Windows / Linux                        |
| `Cmd + K`    | macOS                                  |
| `J` then `K` | Joomla native keyboard shortcut system |

---

## Notes

* The command palette is loaded by the system plugin inside Joomla Administrator.
* The administrator component provides dashboard information and quick access to plugin settings.
* The dashboard hint can be disabled from the plugin configuration.
* Search results are filtered according to administrator access permissions.
* JT Admin Finder is designed for Joomla 6.1+, PHP 8.3+, and the Atum administrator template.
